# 株式会社サントー コーポレートサイト リニューアル仕様書

## 概要
- **クライアント**: 株式会社サントー
- **業種**: 人材派遣業
- **目的**: 求職者・求人企業双方に安心・信頼感を与えるリニューアル
- **既存サイト**: http://santo-hp.co.jp/
- **技術スタック**: Next.js + Tailwind CSS + shadcn/ui

## デザイン方針
- **テーマ**: シンプル・青基調・コーポレート感
- **禁止事項**: AIっぽいレイアウト（グラデーション過多、浮遊するカード、過剰なアニメーション）
- **目指す印象**: 堅実・信頼・清潔感のある日本の人材派遣会社
- **ロゴ**: クライアント支給（配置箇所：ヘッダー左上）
- **写真素材**: 一部クライアント支給、不足分はプレースホルダー

## ターゲットユーザー
- 20〜40代の男女
- **求職者**: 求人内容を主に確認（仕事を探している人）
- **求人企業**: 会社情報を主に確認（人材を探している企業）
- 閲覧時間想定: 2〜3分 → 知りたい情報にすぐ辿り着ける構成

## ページ構成

### 1. トップページ（ホーム）
- ヒーローセクション（キャッチコピー + メインビジュアル）
- 求職者向け・企業向けの2つの導線（目立つCTA）
- サービスの特徴（3つ程度、簡潔に）
- 新着情報 / お知らせ
- **労働者派遣事業に係る情報公開**リンク
- **株式会社アドバンスフローが運営する派遣会社おすすめランキング**（目立つバナー）

### 2. 会社概要
- 代表挨拶
- 企業理念
- 会社情報テーブル（社名、所在地、設立、資本金、代表者、事業内容、許可番号など）
- 沿革

### 3. 仕事をお探しの方（求職者向け）
- サントーで働くメリット
- 派遣の流れ（ステップ形式）
- 求人例・職種紹介
- よくある質問（FAQ）
- 応募CTAボタン

### 4. 人材をお探しの企業様（求人企業向け）
- サントーの人材派遣サービスの強み
- サービスの流れ（ステップ形式）
- 対応業種・職種
- お問い合わせCTA

### 5. アクセス
- 所在地情報
- Google Mapsの埋め込み
- 交通アクセス案内

### 6. お問い合わせ
- お問い合わせフォーム（名前、メール、電話、種別、内容）
- 電話番号
- 営業時間

### 7. 多言語ページ
- **英語 (English)**
- **ポルトガル語 (Português)**
- **スペイン語 (Español)**
- 簡易的な構成でOK（会社概要 + 求人案内 + お問い合わせ）
- 外国籍求職者に安心感を与える目的
